# 🖤✨ Luxury Golden AI Chatbot (Frontend Only)

A high-end, responsive AI chatbot interface built using **HTML, CSS, and JavaScript**, designed with a **premium Golden & Black aesthetic**.

This project focuses on **UI/UX excellence**, modular architecture, and frontend simulation of AI interaction.

## 🎯 Features

### 💬 Chat Interface

* Real-time message display
* User (right) & Bot (left) message bubbles
* Auto-scroll to latest message

### ⚡ Typing Simulation

* Animated typing indicator
* Delayed bot responses for realism

### 🎯 Quick Actions

* Formal Wear
* Bridal
* Alterations

👉 Click to instantly interact with chatbot

---

### 💾 Persistent Chat (localStorage)

* Messages saved in browser
* Chat remains after refresh

---

### 🎨 Premium UI/UX

* Golden & Black luxury theme
* Glassmorphism effects
* Smooth animations & transitions
* Responsive design (mobile + desktop)

---

## 🧱 Tech Stack

* CSS3
* JavaScript (ES6+)

## 🧠 Architecture Overview

* **app.js** → Handles events & initialization
* **chat.js** → Controls chat flow
* **ui.js** → DOM rendering & animations
* **bot.js** → Mock AI responses
* **storage.js** → localStorage management

👉 Fully modular and scalable
